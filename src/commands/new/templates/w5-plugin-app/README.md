# W5 Plugin App Template

This project was generated from Acton's `w5-plugin` template with the
TypeScript app scaffold enabled. It includes a Wallet V5 extension contract
(`SimpleExtension`), Tolk tests and wrappers under `contracts/`, generated
TypeScript wrappers in `wrappers-ts/`, and a Vite-based React app in `app/`
that talks to a connected Wallet V5 through TonConnect.

## Layout

- `contracts/src` contains the extension contract, shared Tolk types, and the
  vendored Wallet V5 sources under `contracts/src/walletv5/`.
- `contracts/tests` contains integration tests.
- `contracts/wrappers/*.gen.tolk` contains Tolk wrappers generated by
  `acton wrapper` and used by tests and scripts.
- `contracts/wrappers/WalletV5Contract.tolk` is a hand-written wrapper for the
  vendored Wallet V5.
- `contracts/scripts` contains the deploy, install, and delete scripts.
- `wrappers-ts/SimpleExtension.gen.ts` and `wrappers-ts/WalletV5.gen.ts` are
  the generated TypeScript wrappers consumed by the app.
- `app/` contains the React + Vite frontend.
- `package.json`, `tsconfig.json`, and `vite.config.ts` configure the app
  toolchain.
- `package-lock.json` pins the npm dependency tree for reproducible installs.

## Install

```bash
npm ci
```

## Commands

```bash
acton build
acton test
npm run build
npm run typecheck
npm run fmt:check
npm run dev
```

## Notes

- `acton build` compiles the contracts using `contracts/src/SimpleExtension.tolk`
  and the vendored Wallet V5.
- `acton wrapper SimpleExtension` regenerates
  `contracts/wrappers/SimpleExtension.gen.tolk`.
- `acton wrapper SimpleExtension --ts` regenerates
  `wrappers-ts/SimpleExtension.gen.ts`.
- `npm run build` runs the contract build and the frontend build.
- `npm run test` delegates to `acton test`.
- The app uses Vite, npm, shadcn-style UI primitives, Tailwind CSS, and
  `@ton/appkit-react` for the wallet UI flow.
- Copy `.env.example` to a local `.env` for Toncenter keys. Both Acton CLI and
  the Vite app read `TONCENTER_MAINNET_API_KEY` and
  `TONCENTER_TESTNET_API_KEY`; Vite allows the `TONCENTER_` prefix via
  `envPrefix` in `vite.config.ts`.
