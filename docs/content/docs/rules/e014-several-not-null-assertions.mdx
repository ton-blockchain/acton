### What it does
Detects repeated not-null assertions like `foo!!`.

### Why is this bad?
Repeating `!` is redundant: after the first not-null assertion the value is already non-null.

### Example
```tolk twoslash
val x = foo!!;
//         ^^ E014: several not-null assertions in a row
```

Use instead:
```tolk
val x = foo!;
```
