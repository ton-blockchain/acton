### What it does
Detects negated type checks written as `!(a is T)`.

### Why is this bad?
Tolk has a dedicated `!is` operator, which is clearer than negating `is`.

### Example
```tolk twoslash
if (!(value is int)) {
//  ^^^^^^^^^^^^^^^ E022: negated `is` type check can use `!is`
    return;
}
```

Use instead:
```tolk
if (value !is int) {
    return;
}
```
