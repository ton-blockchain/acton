## Metadata

- `Code`: `E006`
- `Rule`: `unused-import`
- `Status`: Stable since `v0.0.1`
- `Quick fix`: sometimes available

### What it does
Checks for imports that are never used in the file.

### Why is this bad?
Unused imports clutter the code and increase compilation time.

### Example
```tolk
import "other.tolk";

fun main() {
    println("hello");
}
```

Use instead:
```tolk
fun main() {
    println("hello");
}
```
