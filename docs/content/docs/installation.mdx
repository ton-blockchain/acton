## Overview

Acton ships as a single, dependency-free executable. You can install it via script,
Homebrew, or download single binary from releases.

<Callout>
    After installation, verify with `acton --version`
</Callout>

## Installation

<Callout type="warning">
  Before public release this method is not working, please use [Direct Downloads](#direct-downloads)
</Callout>

<Tabs items={["macOS", "Linux"]}>
    <Tab value="macOS">

        ```bash
        curl -fsSL https://raw.githubusercontent.com/i582/acton/main/install.sh | bash
        ```

    </Tab>
    <Tab value="Linux">

        ```bash
        curl -fsSL https://raw.githubusercontent.com/i582/acton/main/install.sh | bash
        ```

    </Tab>
</Tabs>

To check that Acton was installed successfully, open a new terminal window and run:

```bash
acton --version
```

You should see the version number of Acton.

<Callout type="warning">
    If you've installed Acton but are seeing a `command not found` error, you
    may have to manually add the installation
    directory to your `PATH`.
</Callout>

<Accordions>
    <Accordion title="Add Acton to your PATH">
        <Tabs items={["macOS & Linux"]}>
            <Tab value="macOS & Linux">
                <Steps>
                    <Step title="Determine which shell you're using">
                        ```bash
                        echo $SHELL
                        # /bin/zsh or /bin/bash or /bin/fish
                        ```
                    </Step>

                    <Step title="Open your shell configuration file">
                        * For bash: `~/.bashrc`
                        * For zsh: `~/.zshrc`
                        * For fish: `~/.config/fish/config.fish`
                    </Step>

                    <Step title="Add the Acton directory to PATH">
                        Add this line to your configuration file:

                        ```bash
                        export PATH="PATH_TO_ACTON_BIN:$PATH"
                        ```
                    </Step>

                    <Step title="Reload your shell configuration">
                        ```bash
                        source ~/.bashrc # or ~/.zshrc
                        ```
                    </Step>
                </Steps>
            </Tab>
        </Tabs>
    </Accordion>
</Accordions>

## Updating

<Callout type="warning">
  Before public release updating is not working without explicit GitHub token, please update manually
</Callout>

Once Acton is installed, you can use the built-in `acton up` command to update to the latest version or switch between versions.

### Update to latest stable

To update to the latest stable version, simply run:

```bash
acton up
```

### List available versions

To see all available versions, including the trunk release:

```bash
acton up --list
```

### Install a specific version

To install a specific version:

```bash
acton up 0.1.5
```

### Switch to trunk

If you want to try the latest features before they are officially released, you can install the trunk version:

```bash
acton up --trunk
```

## Uninstall

To uninstall Acton binaries/cache, remove the `.acton` directory in your home folder:

```bash
rm -rf ~/.acton
```

If you also want to remove global wallets and libraries, remove the global config directory too:

```bash
rm -rf ~/.config/acton
```
