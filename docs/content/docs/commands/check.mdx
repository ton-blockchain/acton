---
title: "Checking Tolk contracts"
description: "Lint Tolk files and contracts"
---

The `acton check` command runs the Tolk linter on your project. By default, it checks all contracts from `Acton.toml` and any `*.test.tolk` files in the workspace.

## Usage

```bash
acton check
acton check <contract-id>
acton check <path-to-file.tolk>
```

## Options

| Option                     | Type   | Description                                                 |
|:---------------------------|:-------|:------------------------------------------------------------|
| `--fix`                    | Flag   | Automatically apply available fixes (plain output only).    |
| `--output-format <FORMAT>` | String | Output format: `plain`, `json`, or `sarif`.                 |
| `--output-file <PATH>`     | String | Write JSON/SARIF output to file (not allowed with `plain`). |
| `--explain <CODE>`         | String | Explain a rule.                                             |

For setup and configuration, see [Linting](/docs/linting).
For the full rules catalog, see [Linter Rules](/docs/linting/rules).

## Behavior Notes

- `--fix` applies edits only in `plain` output mode.
- With `--output-format json` or `--output-format sarif`, diagnostics include fix suggestions, but files are not changed.
- Output format is resolved in this order: CLI `--output-format`, then `[lint].output-format` from `Acton.toml`, then default `plain`.
- `--output-file` cannot be used with `plain` format.
- `acton check <path-to-file.tolk>` runs the file in single-file mode with test-style lint settings. In this mode, `E014 (acton-import-in-contract)` is relaxed.
- `acton check <contract-id>` keeps normal contract lint settings from `Acton.toml` (including `E014`).

## Examples

```bash
# Check the whole project
acton check

# Check a specific contract by id from Acton.toml
acton check counter

# Check a single file
acton check counter.tolk

# Print JSON to stdout
acton check --output-format json

# Write SARIF report to file
acton check --output-format sarif --output-file .acton/reports/lint.sarif
```
