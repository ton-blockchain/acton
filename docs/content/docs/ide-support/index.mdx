Official support for Acton development is available for most popular IDEs through extensions that integrate deeply with the Acton toolchain.

Acton have first-class support in the following editors:

- **[VS Code and VS Code-based editors](vscode)** (Cursor, Windsurf, VSCodium, etc.)
- **[JetBrains IDEs](jetbrains)** (IntelliJ IDEA, WebStorm, GoLand, PyCharm, CLion, etc.)
