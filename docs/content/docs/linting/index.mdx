This section contains guides and reference documentation for Acton's linter.

`acton check` helps catch common issues and enforce best practices in Tolk code.
It supports project-wide checks, target-specific checks, automatic fixes, JSON output, and SARIF reports.

## Basic Usage

```bash
acton check
acton check counter
acton check contracts/counter.tolk
```

## Common Workflows

```bash
# Apply automatic fixes where available
acton check --fix

# Explain a specific diagnostic code
acton check --explain E002

# Machine-readable output
acton check --json
acton check --sarif .acton/reports/lint.sarif
```
