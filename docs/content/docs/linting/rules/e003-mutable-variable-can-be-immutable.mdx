### What it does
Checks for variables that are declared as mutable (`var`) but are never mutated.

### Why is this bad?
Using `val` instead of `var` makes the code clearer by signaling that the variable's value will not change.

### Example
```tolk
fun main() {
    var x = 1;
    println(x);
}
```

Use instead:
```tolk
fun main() {
    val x = 1;
    println(x);
}
```
