### What it does
Detects duplicated conditions in `if / else if` chains.

### Why is this bad?
Repeating the same condition in one conditional chain usually means a
copy-paste bug and can make branches unreachable.

### Example
```tolk
if (a < 1) {
    return 1;
} else if (a > 4) {
    return 2;
} else if (a > 4) {
    return 3;
}
```
