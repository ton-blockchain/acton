Replay a transaction trace by hash

Usage: acton[EXE] retrace [OPTIONS] <HASH>

Arguments:
  <HASH>  Transaction hash in hex format to retrace

Options:
      --color <WHEN>          Control when to use colored output [default: auto] [possible values: auto, always, never]
      --net <NET>             Network to use
      --manifest-path <PATH>  Path to Acton.toml
      --verbose               Show full cell hex instead of hashes in out actions
      --logs-dir <LOGS_DIR>   Directory to save VM and executor logs
      --project-root <PATH>   Path to project root
      --contract <CONTRACT>   Contract name from Acton.toml used to build a source-level trace for the transaction
  -h, --help                  Print help

Debugging:
      --debug                    Enable source-level debugging for the retraced transaction; requires --contract
      --debug-port <DEBUG_PORT>  Debug server port

Run 'acton help retrace' for more detailed information.
