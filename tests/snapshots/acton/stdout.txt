TON blockchain development tool

Usage: acton [OPTIONS] <COMMAND>

Commands:
  init         Initialize a new project in the current directory
  new          Create a new project in a specified directory
  wallet       Manage wallets
  test         Execute tests in file or directory
  wrapper      Generate wrapper and optionally stub test file for a contract
  script       Execute a Tolk script file
  build        Build the specified contract or all contracts
  run          Run a script defined in Acton.toml
  compile      Compile a Tolk file
  disasm       Disassemble TVM bitcode to human-readable TASM
  verify       Verify contract source code on verifier.ton.org
  check        Check Tolk files in the project for errors
  retrace      Retrace a transaction by its hash
  library      Manage TON libraries
  litenode     Manage lightweight TON node
  fmt          Format Tolk source files
  ls           LSP server for the TON languages and technologies
  up           Manage Acton versions
  completions  Generate shell completions for selected shell
  help         Print this message or the help of the given subcommand(s)

Options:
      --color <WHEN>          Control when to use colored output [default: auto] [possible values: auto, always, never]
      --manifest-path <PATH>  Path to Acton.toml
  -h, --help                  Print help
  -V, --version               Print version
