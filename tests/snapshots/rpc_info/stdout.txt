Show remote account information and decode storage when ABI is known

Usage: acton[EXE] rpc info [OPTIONS] <ADDRESS>

Arguments:
  <ADDRESS>  Contract address in friendly or raw format

Options:
      --block-number <BLOCK_NUMBER>  Masterchain block seqno to query account state at
      --color <WHEN>                 Control when to use colored output [default: auto] [possible values: auto, always, never]
      --manifest-path <PATH>         Path to Acton.toml
      --net <NET>                    Network to query (defaults to testnet). Supported values: mainnet, testnet, localnet, custom:<name>
      --project-root <PATH>          Path to project root
  -h, --help                         Print help
